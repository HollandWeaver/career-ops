# Daily Application Quota

Track applications submitted per day by match tier. Goal: ~50/day across all tiers.

## Targets

| Tier | Match % | Label | Daily Target | % of Total |
|------|---------|-------|-------------|------------|
| T1 | 96–100% | One-to-one match | 25 | 50% |
| T2 | 91–95% | Strong fit | 15 | 30% |
| T3 | 86–90% | Good fit | 5 | 10% |
| T4 | 80–85% | Reach | 5 | 10% |
| **Total** | | | **50** | |

## Rules

- Fill T1 first each day before applying to T2. Fill T2 before T3/T4.
- Max **2 applications per company** unless roles are closely related (same team, very similar function).
- If T1 is empty on a given day, shift 10 slots to T2.
- <80% match → do not apply regardless of other factors.
